Help

How to use LocalEssentials + troubleshooting + privacy

How it works

  1. Enter a UK postcode OR use your location
  2. Choose essentials + radius
  3. View results and open Directions
  4. Click to Save any places to favourite

Troubleshooting

  • Invalid postcode: check format (e.g., SW1A 1AA)
  • Location denied: enter postcode instead
  • No results: increase radius or change category

Privacy

  • No accounts required
  • Location is used only to search nearby essentials
  • Saved settings/favourites are stored locally in your browser (LocalStorage)
  • You can clear data by clearing browser storage
LocalEssentials • Static site (HTML/CSS/JS) • Uses Postcodes.io + OpenStreetMap data
